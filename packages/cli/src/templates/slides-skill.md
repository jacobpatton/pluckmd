# harvest slides

Generate dark-themed HTML slides (5-12 slides with SVG diagrams) from Wiki entries.

Usage: `/harvest-slides <wiki entry path or concept name>`

Design: #0f0f1a bg, #e94560 accent, #f5c842 gold, inline SVG, keyboard arrows and swipe support.
Saved to wiki/slides/<name>.html.
